# mycompiler

Minimal Mu-like compiler that supports:
- `name : int = <int>;`
- `name : int;`
- `name := <int>;`
- `print(name);` or `print(123);`
- `print("%\n", name);`

Build:
```
mkdir -p build
cd build
cmake ..
cmake --build .
```

Run:
```
./mucc path/to/file.mu -o out
./out
```
